# boilerplate golang clean architechture

## Requirements

 - Golang version 1.7+
 - Glide; click https://github.com/Masterminds/glide
 - other needed modules

## Installing

 - Clone project
 - Install Glide https://github.com/Masterminds/glide
 - Install Dependecies
    ```console
    $ glide install
    ```

## Run 
 - Run the program
   ```console
   $ go build && ./boilerplate-go-cleanarch
   ```
